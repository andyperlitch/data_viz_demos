gRaphael

Specs

  • minified size: ~120kb (Raphael, gRaphael, graph type modules)
  • license: MIT
  • customizable: yes
  • native object constancy: no

Pros:

  • simple API
  • relatively lightweight
  • broad browser support

Cons:

  • minimal feature list
  • updating data, i.e. adding new points is not natively supported
